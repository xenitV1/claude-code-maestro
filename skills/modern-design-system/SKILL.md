# Modern Design System

> Principles for premium, aesthetic web interfaces.

---

## 1. 2025 Visual Style Principles

### Style Selection

| Style | Best For | Effect |
|-------|----------|--------|
| **Glassmorphism** | Cards, overlays | Depth, elegance |
| **Bento Grid** | Feature showcases, dashboards | Visual interest |
| **Neumorphism** | Interactive elements | Tactile feel |
| **Dark Mode** | Most apps | Modern, premium |

---

## 2. Glassmorphism Principles

### Key Properties

| Property | Value Range | Effect |
|----------|-------------|--------|
| Background | 5-15% opacity | Transparency |
| Blur | 8-24px | Frosted effect |
| Border | 10-20% opacity | Definition |
| Shadow | Subtle | Depth |

### When to Use

- Overlays on images
- Cards with depth
- Modal backgrounds
- Navigation bars

---

## 3. Bento Grid Principles

### Layout Rules

| Element | Span | Use |
|---------|------|-----|
| Hero/Feature | 2x2 | Main content |
| Stats | 1x1 | Metrics |
| Secondary | 2x1 | Supporting |
| Accent | 1x1 | Visual balance |

### Spacing

- Gap: 16-24px between items
- Padding: 24-32px inside cards
- Radius: 0-2px (Keskin/Lüks) veya 16-32px (Yumuşak) → 8px'lik "Sıkıcı Bölge"den KAÇININ.

---

## 4. Animation Principles

### Animation Types

| Type | Duration | Use |
|------|----------|-----|
| Micro | 150-300ms | Buttons, icons |
| Page transition | 300-500ms | Navigation |
| Scroll reveal | 400-700ms | Content entry |
| Hover | 200-300ms | Interactive feedback |

### Animation Guidelines

- Ease-out for entries
- Ease-in-out for transitions
- Spring for playful interactions
- Subtle Y-translation for depth

## 5. Color System Principles

### Dark Mode Palette

| Layer | Purpose |
|-------|---------|
| Background | #0a0a0a - #111111 |
| Surface | #1a1a1a - #262626 |
| Text Primary | #fafafa |
| Text Secondary | #a3a3a3 |
| Accent | Vibrant gradient |

### Gradient Usage

| Type | Use |
|------|-----|
| Linear | Headlines, buttons |
| Radial | Backgrounds, glows |
| Mesh/Aurora | Hero sections |

## 6. Typography Principles

### Scale (1.25 ratio)

| Level | Use |
|-------|-----|
| Hero | 48-64px, tight leading |
| H1 | 36-48px |
| H2 | 24-30px |
| Body | 16-18px |
| Small | 12-14px |

### Font Pairing

| Role | Examples |
|------|----------|
| Display | Cal Sans, Clash Display |
| Body | Inter, Outfit |
| Code | JetBrains Mono, Fira Code |

## 7. Premium Touches

| Element | Implementation |
|---------|---------------|
| Glow effects | Box-shadow with accent color |
| Gradient text | Background-clip text |
| Grid patterns | Subtle background |
| Noise texture | Very low opacity overlay |
| Border gradients | Pseudo-elements |

---

## 8. Component Checklist

### Cards
- [ ] Rounded corners (0-2px Sharp veya 16-32px Ultra-Soft)
- [ ] Subtle border
- [ ] Hover state with Y-translate
- [ ] Gradient overlay on hover

### Buttons
- [ ] Gradient or solid accent
- [ ] Scale on hover/tap
- [ ] Glow effect (primary)
- [ ] Clear disabled state

### Hero Sections
- [ ] Large typography
- [ ] Gradient text
- [ ] Background pattern/mesh
- [ ] Staggered animations

---

## 9. Anti-Patterns

| ❌ Don't | ✅ Do |
|----------|-------|
| Generic blue-purple gradient | Custom brand gradient |
| Same card style everywhere | Varied layouts |
| Static UI | Micro-animations |
| Default Tailwind colors | Curated palette |
| System fonts only | Custom font pairing |
| White background everywhere | Subtle patterns |

---

## 10. Quality Checklist

- [ ] Custom color palette (not defaults)
- [ ] Gradient accents used
- [ ] Proper dark mode contrast
- [ ] Custom typography scale
- [ ] Micro-animations on interactions
- [ ] Scroll-triggered reveals
- [ ] Generous whitespace
- [ ] Visual hierarchy clear

---

> **Remember:** Premium design is about restraint and intentionality. Less is more, but every detail matters.
